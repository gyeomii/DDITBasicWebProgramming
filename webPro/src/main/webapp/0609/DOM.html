DOM (Document Object Model, 문서 객체 모델)
	
	- HTML페이지(=웹문서)를 인식할 수 있도록 구성된 문서 객체 모델
	- Document와 관련된 객체 집합으로
		프로그래밍 언어가 DOM구조에 접근할 수 있는 방법 제공
		>> 문서의 구조, 스타일, 내용 등의 변경을 도움

	- 자바스크립트를 통해 동적 변경이 가능하고 접근 방식은
		getElement... , querySelect ... 들을 사용	

	DOM Tree 구조 - 관련 NODE
	
- Document(9)  //자주 씀
  ㄴDocumentType(10)				<!DOCTYPE HTML>
  ㄴElement(1)  //자주 씀			<textarea>, <p>, <font>
    ㄴAttribute(2)  //자주 씀			<textarea rows="30" cols="40">
    ㄴEntity(6)					&lt;text&gt;
    ㄴEntityReference(5)			&lt;text&gt;
    ㄴText(3)  //자주 씀			<p>text</p> 		==> 공백 데이터 포함
  ㄴComment(8)					<!-- 주석 -->
  ㄴDCATASection(4)				<![CDATA[]]>		==> 특수문자,비교연산자등을 문자열로 인식하도록 처리
  ㄴNotation(7)					<font color="#ffff00" size="10px"> ==> #ffff00, 10px
