	JavaScript코드는 웹 브라우저가 모든 문서 요소들을 로드한 뒤
	실행되어야 오류가 발생하지 않는다
	(=생성되기 전 요소에 속성을 추가하려고 할때 에러 발생)
	
	해결방법
	  - window객체의 load 사용
	  	문서의 모든 콘텐츠(images, script, css, etc...)가 로드된 후 
	  	발생하는 이벤트로서 문서의 모든 요소 로딩 후 실행되므로 (상대적으로) 느림
	  - document객체의 DOMContentLoaded 사용
	  	html과 script가 로딩된 시점에서 발생하는 이벤트로서 
	  	load보다 먼저 발생되므로 실행이 더 빠르다
