안녕 만나서 반갑다숨겨진텍스트 
	요소에 값을 읽어오거나 설정하는 키워드
	
	value 
		1. 입력양식 <input>의 초기값 명시에 사용
		2. 입력양식 type에 따라 용도가 변경
			- button, reset, submit : 버튼 내 텍스트 정의
			- text, password, hidden : 입력 필드 초기값 정의
			- checkbox, image, radio : 필드 선택시 서버로 제출되는 값 정의
	
	innerHTML
		읽어오는 값에 태그를 포함한다 (보안에 취약한 단점이 존재)
		XSS : Cross-site Scripting
		
		HTML5에서는 innerHTML과 함께 삽입된 태그를 실행하지 않도록 지정했으며
		보안점검을 거치는 프로젝트의 경우 코드거부가능성이 있음
		
		특별한 경우를 제외하고 사용을 권장하지 않음
				
	innerText
		element의 속성
		사용자에게 보여지는 텍스트의 값을 읽어옴
		연속되는 공백은 무시(= 하나의 공백으로 처리)
		원시 텍스트의 랜더링된 상태(=스타일 적용 후)를 보여줌
	
	textContent
		node의 속성
		해당 노드의 텍스트값(=원시 텍스트)을 읽어옴
		표현된 공백을 그대로 처리하며,
		숨겨진 텍스트도 출력
