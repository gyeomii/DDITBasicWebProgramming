변수 키워드 3종 (var, let, const)
	
	var
		- scope : 함수 범위
		- 변수 호이스팅 발생 (=선언 이전 참조를 가능하게 하여
								비정상적 흐름의 코드 작성을 가능하게 함)
		- 변수 선언시 키워드 생략 허용(=의도하지 않은 변수의 전역화)
		- 재선언, 재할당 가능
		
	>> 코드의 복잡도를 높이고 오류를 발생시키는 등 다수의 문제를
	발생시켜 이를 보완하고자 ES6에서 let, const 도입
		
	let
		- scope : 블록('{}') 범위
		- 변수 사용전까지 초기화되지 않음
		- 변수 재선언 불가
		- 변수 재할당 가능
		- 변수 호이스팅 불가
		
	const
		- scope : 블록('{}') 범위
		- 변수 사용전까지 초기화되지 않음
		- 변수 재선언 및 재할당 불가
