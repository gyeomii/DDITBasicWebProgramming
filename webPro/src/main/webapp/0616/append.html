추가 메소드
    append, appendTo >> 선택요소의 안쪽 뒤에 붙여넣음
    prepend, prependTo >> 선택요소의 안쪽 앞에 붙여넣음
    before, insertBefore >> 선택요소의 바깥쪽 앞에 붙여넣음
    after, insertAfter >> 선택요소의 바깥쪽 뒤에 붙여넣음

    
    $(부모요소).append(,prepend, before, after)(자식요소)
    $(자식요소).appendTo(,prependTo, insertBefore, insertAfter)(부모요소)

    *appendChild는 jQuery객체에 사용 불가, Dom객체 - node로 생성된 객체만 사용
    *부모노드.appendChild(자식요소)

    *appendChild() 와 append(), appendTo() 차이점

     - appendChild() :  노드객체만 사용 가능
                     : 한번에 하나의 노드만 추가 가능
                     : return값을 반환

     - append(To)()  : 노드객체나 DomString(text) 사용가능
                     : 한번에 여러개의 자식요소 추가 가능 
                     : return값을 반환하지 않음
